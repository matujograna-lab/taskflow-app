============================================
TASKFLOW - Task Management Application
============================================

Project Structure:
------------------
taskflow-project/
├── index.html    (Main HTML file)
├── styles.css    (All styles and animations)
├── app.js        (React components and logic)
└── README.txt    (This file)

How to Run:
-----------
1. Make sure all three files are in the same folder
2. Open index.html in a web browser
   - Double-click the file
   - OR use Live Server in VS Code
3. Create an account or login to start

Features:
---------
✅ User Authentication (Login/Signup)
✅ Create, Edit, Delete Tasks
✅ Kanban Board (To Do, In Progress, Completed)
✅ Search Tasks
✅ Filter by Priority
✅ Dark Theme Design
✅ Local Storage - Tasks save automatically
✅ Due Date & Time with validation

Demo Credentials:
-----------------
Create your own account - no demo credentials needed!

Tech Stack:
-----------
- HTML5
- CSS3
- React 18
- JavaScript (ES6+)
- LocalStorage for data persistence
